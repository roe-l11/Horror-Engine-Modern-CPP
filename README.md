# 🧟 Horror Engine v1.0 (Modern C++ & SDL2)

مستودع احترافي ومفتوح المصدر لمحاكاة وبناء محرك ألعاب رعب كلاسيكي وثنائي الأبعاد (2D Platformer) مستوحى من ألعاب الجيل القديم، تم بناؤه باستخدام لغة **C++ الحديثة** والاعتماد على مكتبة **SDL2** لإدارة الرسوميات والصوتيات والفيزياء بشكل كامل وسلس.

---

## 🗺️ تخطيط وهيكلية المحرك (Engine Architecture)
يعتمد المحرك على بنية معمارية منظمة لإدارة الأطر (Game Loop)، معالجة مدخلات اللاعب، تحريك الأعداء بالذكاء الاصطناعي، وحسابات الفيزياء والجاذبية وفحص التصادم بشكل متزامن.

## 📜 الكود البرمجي الكامل للمحرك (`main.cpp`)
الكود متاح بالكامل داخل هذا المستودع في ملف `main.cpp` للتعديل والتطوير. تم تنظيف المنظومة بالكامل وتعديل الكود برمجياً لاستبدال أي مسميات قديمة بـ "البلورات السحرية المتوهجة" لتكون هي عنصر الطاقة والتشغيل الرئيسي للاعب.

## 🎨 دليل إدارة الموارد الحقيقية والأصوات (Assets Guide)
* 🎵 الأصوات: إضافة ملف `music.mp3` للموسيقى الخلفية، وملف `hit.wav` للمؤثرات الصوتية.
* 🖼️ الرسوميات: إضافة `player.png` و `zombie.png`. وفي حال احتواء الصور الأصلية المستخرجة من الأرشيف على رموز صلبان قديمة، يجب تعديلها وتغطيتها رقمياً ببلورات حمراء أو خضراء مضيئة قبل الرفع ليتناسق المظهر مع الكود النظيف.

## 🛠️ التثبيت والتشغيل (Installation & Compilation)
```bash
sudo apt-get install libsdl2-dev libsdl2-image-dev libsdl2-mixer-dev
g++ main.cpp -o horror_game -lSDL2 -lSDL2_image -lSDL2_mixer
./horror_game
```
